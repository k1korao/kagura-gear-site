# Kagura Gear

Premium ecommerce website for Kagura Gear, a dark Japanese-inspired gaming gear brand.

Slogan: **Precision Meets Ritual**

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- npm
- Vercel-ready deployment
- Shopify Buy Button / Shopify Checkout placeholders

## Local Development

```bash
npm install
npm run dev
```

## Shopify Payments

This project does not process credit cards and does not store payment data.

Use Shopify Checkout or Shopify Buy Button for payment. Product cards and product detail pages include a clear placeholder component:

```txt
src/components/ShopifyBuyButtonSlot.tsx
```

When your Shopify products are ready:

1. In Shopify Admin, create a Product Buy Button.
2. Copy the embed code.
3. Replace the placeholder area inside `ShopifyBuyButtonSlot.tsx`.
4. Keep checkout handled by Shopify.

Future headless Shopify placeholders are listed in `.env.example`.

## Support Email and Contact Form

The production support inbox is:

```txt
support@kaguragear.com
```

The website includes direct `mailto:` links so customers can email Kagura Gear
immediately. The contact page also includes a server route at `/api/contact`.
To make the web form send email directly from Vercel, connect an email provider
and add these Vercel environment variables:

```txt
RESEND_API_KEY=your-resend-api-key
CONTACT_FROM_EMAIL=Kagura Gear <support@kaguragear.com>
CONTACT_TO_EMAIL=support@kaguragear.com
NEWSLETTER_FROM_EMAIL=Kagura Gear <support@kaguragear.com>
NEWSLETTER_NOTIFY_EMAIL=support@kaguragear.com
```

Do not add real secrets to GitHub. Add them only in Vercel Project Settings >
Environment Variables. If the provider is not connected yet, the form shows a
safe direct-email fallback instead of pretending the message was sent.

The newsletter form posts to `/api/newsletter`. After a customer signs up and
confirms consent, the site sends an automatic welcome email with recommended
launch products and notifies the support inbox. For long-term marketing lists,
scheduled campaigns, and unsubscribe management, connect Shopify Email, Klaviyo,
Mailchimp, or another email marketing platform instead of using the website as a
customer database.

## Vercel Deployment

1. Push this project to GitHub.
2. Open Vercel.
3. Import the GitHub repository.
4. Use the default Next.js settings:
   - Install Command: `npm install`
   - Build Command: `npm run build`
   - Output Directory: leave blank/default
5. Deploy.

## Connect kaguragear.com on Vercel

In the Vercel project:

1. Go to Settings > Domains.
2. Add `kaguragear.com`.
3. Add `www.kaguragear.com`.
4. Vercel will show the DNS records required for each domain.

Because the domain was purchased through Shopify, copy the records from Vercel into Shopify domain DNS settings.

Typical setup:

- Root domain `kaguragear.com`: A record using the IP shown by Vercel.
- `www.kaguragear.com`: CNAME record using the target shown by Vercel.

Use the exact values Vercel displays. DNS can take 24 to 48 hours to fully propagate.

## Google Search Console

After the Vercel deployment is live and the domain works:

1. Open Google Search Console.
2. Add a Domain property for `kaguragear.com`.
3. Verify ownership using the TXT record Google provides.
4. Add that TXT record in Shopify DNS.
5. After verification, open Sitemaps.
6. Submit:

```txt
https://kaguragear.com/sitemap.xml
```
